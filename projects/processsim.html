Brass Drawing Process Optimisation

Abaqus and Taguchi-based optimisation of a brass alloy drawing process.

Project Overview

Context: MSc manufacturing simulation project focused on the optimisation of a brass alloy drawing process using numerical simulation and statistical analysis.

Problem: The project investigated how process parameters affected drawing force, tensile stress, springback, tool behaviour, energy profile and process stability.

My Role: I contributed to the Abaqus finite element simulations, Taguchi experimental design, parameter comparison, result interpretation and optimisation reasoning for process performance and tool behaviour.

Duration: MSc Simulation of Technological Processes coursework project.

Tools: Abaqus, finite element analysis, Taguchi method, DOE, regression analysis, mesh refinement, stress analysis, force analysis, springback analysis and process optimisation.

Technical Evidence

Outcome

The study showed that drawing force was mainly influenced by friction factor and initial bar diameter, while tensile stress depended mostly on initial diameter. Mesh refinement changed the preferred parameter combination, highlighting the importance of simulation accuracy in manufacturing process optimisation.

Recruiter Value

CAE capability: demonstrates finite element modelling for a manufacturing process.

Materials/process knowledge: links forming parameters to stress, force, springback and tool behaviour.

Decision-making: uses DOE, regression and mesh refinement to support engineering conclusions.
